# 🚀 Java System Design (LLD)

> A complete Low-Level Design (LLD) repository built using Java, containing design patterns, UML diagrams, mini-projects, interview notes, exercises, and real-world examples.

## 📖 About

This repository is my journey of mastering **Low-Level Design (LLD)** using Java.

Instead of only learning theory, every design pattern is implemented with:

- ✅ Real-world Mini Project
- ✅ UML Diagrams
- ✅ Interview Notes
- ✅ Practice Exercises
- ✅ Real-world Examples
